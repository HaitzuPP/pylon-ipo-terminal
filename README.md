# Pylon IPO Terminal

Static multi-page terminal tracking Hong Kong (HKEX) and Australia (ASX) IPOs.

Pages: Overview, Watchlist (confirmed vs provisional), Recent / Price (IPO vs current price),
ASX (full list), HK (full list). AI / Healthcare / Robotics names are flagged.
Click any row for a company overview (core business, top-line revenue, founder, sources).

Data is a 26 Jun 2026 snapshot compiled from HKEX/ASX disclosures and press reporting.
Not investment advice.
